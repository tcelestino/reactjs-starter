# React.js Stater

My setup for start project with React.js

## Dependency

  - gulp
  - gulp-react
  - gulp-load-plugins
  - gulp-processhtml

## Install

```bash
npm install gulp -g
```

node modules

```bash
npm install
```
bower components

```bash
bower install
```

## Usage

### Dev

```bash
gulp
```
### Build

```bash
gulp build
```
