# React.js Starter

My setup for start project with React.js

## Install


```bash
git clone git@github.com:tcelestino/reactjs-starter.git folder-project
```

```bash
cd folder-project
```

```bash
npm install && bower install
```
_optional: sudo_

## Usage

### Dev
```bash
gulp
```

Or

```bash
gulp watch-dev
```

### Build
```bash
gulp build
```
