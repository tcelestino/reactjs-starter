# React.js Stater

My setup for start project with React.js

## Dependencies
  * connect
  * serve-static

## Dev dependencies
  * gulp
  * gulp-react
  * gulp-load-plugins
  * gulp-processhtml
  * gulp-jshint
  * gulp-order
  * gulp-livereload
  * gulp-uglify
  * gulp-concat

## Install

gulp

```bash
npm install gulp -g
```
node modules

```bash
npm install
```
bower components

```bash
bower install
```

## Starter

```bash
  node start
```
